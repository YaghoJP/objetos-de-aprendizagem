# Sorting Cards

## Público-alvo

O objeto de aprendizagem **SORTING CARDS** é destinado a:

- Estudantes de graduação em Ciência da Computação, especialmente entre o 1º e 3º semestre, que estejam iniciando disciplinas de algoritmos e estruturas de dados.
- Alunos do ensino médio técnico com foco em informática/programação.
- Pessoas interessadas em aprender conceitos fundamentais de algoritmos de ordenação de forma prática e lúdica.

Conhecimentos prévios recomendados:

- Noção básica de comparação entre valores (maior/menor).
- Entendimento introdutório do que é um algoritmo.
- Compreensão básica de lógica sequencial.

## Requisitos de Aprendizagem
Ao finalizar a utilização do objeto de aprendizagem SORTING CARDS, espera-se que o estudante seja capaz de:

**Conceitos**
- Compreender o conceito geral de algoritmo de ordenação.
- Diferenciar características entre quatro algoritmos de ordenação trabalhados.
- Entender o funcionamento de comparações e trocas (swaps) nos algoritmos.

**Procedimentos**
- Executar manualmente cada algoritmo utilizando as cartas como representação dos dados.
- Seguir passo a passo a lógica de cada algoritmo de ordenação.
- Comparar a eficiência dos algoritmos com base no número de operações realizadas.
- Representar um vetor/array usando as cartas e manipular elementos simulando operações como troca ou inserção.

**Atitudes**
- Desenvolver o raciocínio lógico de maneira lúdica.
- Trabalhar em equipe na resolução das atividades e discussão dos algoritmos.
- Estimular curiosidade e interesse pelos fundamentos de algoritmos e estruturas de dados.
